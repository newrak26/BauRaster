\# BauRaster WinUI Design V1



\## Designprinzip



BauRaster ist projektzentriert.



Das Bauvorhaben ist das Hauptobjekt.



Alle Informationen werden einem Projekt zugeordnet.



\---



\## Hauptlayout



\### Links (4/12)



\* Suche

\* Filter

\* Projektliste

\* Status

\* Priorität

\* Nächster Schritt



\### Mitte (5/12)



\* Karte

\* Baustellenpins

\* Cluster

\* Gebietsauswahl



\### Rechts (3/12)



\* Projektdetails

\* Fortschritt

\* Letztes Foto

\* Nächste Aktion

\* Schnellaktionen
